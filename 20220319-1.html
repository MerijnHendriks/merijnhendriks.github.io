Development

My requirements for this website were simple:

  • Pages in markedown
  • Mobile and desktop friendly
  • Hostable on github
  • No external databases
  • No template generators
  • No CDN's

Or in short, a simple blog-like website without relying on third-party services. Besides, it's notmywebsite if I didn't re-invent the wheel as least once. I opted for:

  • Writing a small router to fetch articles
  • markedfor transforming markdown to html
  • prismfor code highlighting
  • bootstrapfor quickly putting the page together
